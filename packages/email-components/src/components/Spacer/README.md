# Spacer

The `mjc-spacer` is component for e-mail space between two contents blocks.

```mjml
<mjc-spacer size="small" />
```

## Available props

| Prop name | Type                       | Default  | Required | Description   |
| --------- | -------------------------- | -------- | -------- | ------------- |
| `size`    | `small \| medium \| large` | `medium` | no       | Spacer height |
